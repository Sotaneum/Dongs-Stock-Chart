# Dongs Stock Chart

Dongs Stock Chart :: 웹으로 보는 주식 차트

![cover](./assets/cover.png)

## Info

- 차트 데이터와 제목, SMA 값을 입력하면 차트를 그려줍니다.
- 데모 페이지는 [여기](http://duration.digimoon.net/dev/web/stock-chart/)에 있습니다.

## Finally

- PHP를 공부하면서 구현한 주식 차트입니다.
